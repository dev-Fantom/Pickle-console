# Output Pickel file to console / Pickelファイルをコンソールに出力

Python command to output the contents of a Pickel file to the console

Pickelファイルの内容をコンソールに出力するPythonコマンド

## Environment / 環境
* Python 3.9.1
* Pandas 2.1.0

## Usage / 使い方

```
python -m pickle PKLFILE.pkl
```
